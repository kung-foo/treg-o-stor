# Treg O' Stor

Treg O' Stor is a 100% feature complete clone of the Microsoft Teams client.

![](chonk.png "CHOOOONK!")

## Features

* Single source file
* Lightweight, portable binary
* Secure by default
* No "telemetry" trackers
* 100% Electron free
* Builds on all major platforms
* Optimized for multi-core systems
* Fully Open Source
